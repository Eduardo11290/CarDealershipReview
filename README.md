# Car Dealership Review Platform

A full-stack web application where users can explore car dealerships and leave reviews based on their experiences.

This project was built to practice full-stack development, REST API design, authentication, and cloud deployment.

---

# Live Demo

Frontend: https://car-dealership-review-huzk.vercel.app/ 

Backend API: https://car-dealership-review.onrender.com

---

# Features

- Browse car dealerships
- Read reviews from other users
- Submit your own review and rating
- User authentication (login/logout)
- Full REST API integration
- Cloud deployment

---

# Tech Stack

Frontend
- React
- JavaScript
- Bootstrap

Backend
- Django
- Python
- REST APIs

Database
- MongoDB Atlas

Deployment
- Vercel (Frontend)
- Render (Backend)

---

# Architecture

React frontend communicates with a Django backend through REST API endpoints.  
The backend processes requests and stores data in MongoDB Atlas.
User → React Frontend → Django REST API → MongoDB Atlas
